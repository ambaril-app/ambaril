# Scope Hammer

Acionar quando:

- Slice travou >3 tentativas
- Discoveries excederam appetite da wave
- Rabbit hole inesperado

## 4 Perguntas

1. **"Must-have?"** — Se nao, cortar.
2. **"Ship sem isso?"** — Se sim, mover proxima wave.
3. **"Impacto de nao fazer agora?"** — Avaliar real.
4. **"Problema pre-existente tolerado?"** — Se sim, nao bloqueante.

## Output

```
Scope Hammer — Wave {X}
MUST-HAVE: {lista}
NICE-TO-HAVE → proxima wave: {lista}
CUT: {lista com motivo}
```

Cortar scope nao e fracasso. E disciplina.
