# Common Mistakes — Feature Create Spec

| Mistake                                      | Fix                                                  |
| -------------------------------------------- | ---------------------------------------------------- |
| Codar antes da spec aprovada                 | Spec precisa de review primeiro                      |
| Regras vagas ("handle errors appropriately") | Toda regra especifica e testavel                     |
| Ignorar cross-module deps                    | Verificar dados necessarios de outros modulos        |
| Assumir schema sem checar existente          | Ler schema antes                                     |
| Pular wireframes                             | ASCII wireframes previnem mal-entendidos             |
| §12 com placeholders                         | Risk tier, blast radius, regression concretos        |
| Spec sem §11 Tracking                        | Zero placeholders — modulo sem tracking = incompleta |
